# Trouver-Web-App
Fake News Analysis Web Application ! The application is deployed on Heroku.

Fake News Detection in Python

In this project, we have used various natural language processing techniques and machine learning algorithms to classifty fake news articles using sci-kit libraries from python

Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

Python 3.6

You will also need to download and install below 3 packages after you install either python or anaconda from the steps above
-> Sklearn (scikit-learn)
-> numpy
-> scipy
if you have chosen to install python 3.6 then run below commands in command prompt/terminal to install these packages
pip install -U scikit-learn
pip install numpy
pip install scipy

Dataset 
The data source used for this project is LIAR dataset and one from Kaggle.
